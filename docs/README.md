# docs

Place the demo recording here as `demo.gif` — the root `README.md` references it at
`docs/demo.gif`.

Tips for a good README GIF:
- Keep it short (~10–20s) and loop-friendly.
- Show the flow: start capture → traffic streams in → filter → click a row → inspect
  request/response JSON → Copy as cURL.
- Use a dark theme to match the plugin's UI.
- Aim for ≤ ~5 MB so it loads quickly on GitHub (downscale / drop FPS if needed).
